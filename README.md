# Run Backend

## Install Dependencies
```
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
```

## Run Backend
On powershell
```
.\RunBackend.ps1
```

## Docker
### Backend
run backend
```
docker compose up
```
## Deployment
### Backend
```
docker compose up
```
### Frontend
Run only in deployment server
```
./deployment/frontend.sh
```
